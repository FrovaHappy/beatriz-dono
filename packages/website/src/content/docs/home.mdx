Beatriz-dono es un bot de Discord diseñado para brindar herramientas versátiles y útiles a los servidores. Aquí encontrarás toda la información necesaria para sacarle el máximo provecho.

## ¿Qué necesito para usar Beatriz-dono?

Para comenzar, solo necesitas tener un servidor de Discord y agregar a Beatriz-dono. Una vez agregado, podrás explorar sus diversas funcionalidades.

## ¿Qué encontrarás en esta documentación?

Esta documentación cubre todo lo necesario para utilizar Beatriz-dono, incluyendo:

- Instalación y primeros pasos.
- Configuración de módulos y comandos.
- Información sobre cómo contribuir al desarrollo del bot.

## ¿Cómo contribuir?

Si deseas apoyar el desarrollo de Beatriz-dono, puedes contribuir de varias formas. Visita el [repositorio de GitHub](https://github.com/FrovaHappy/beatriz-dono) para conocer más detalles y encontrar maneras de colaborar.

## ¿Cómo se mantiene Beatriz-dono?

Beatriz-dono es mantenido por un equipo de desarrolladores voluntarios. Además, se financia de manera colaborativa a través de [Ko-fi](https://ko-fi.com/frovahappy).

Beatriz-dono es mantenido por un equipo de desarrolladores voluntarios. Si deseas colaborar, puedes encontrar más información en el [repositorio de GitHub](https://github.com/FrovaHappy/beatriz-dono).
