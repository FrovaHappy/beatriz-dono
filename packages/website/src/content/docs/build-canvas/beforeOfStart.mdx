Algunas cosas que debes tener en cuenta antes de empezar a construir un canvas.

## Como funciona el alineamiento de texto

El funcionamiento es simple, este puedes alinearlo como muestra en la imagen.

![Alineamiento de texto](/assets/docs/alignText.png)

También puedes justificar el texto según como muestra en la imagen.

![Alineamiento de texto](/assets/docs/justifyText.png)

## Como se desplaza las Coordenadas?

Las coordenadas se desplazan positivamente en el eje Y de izquierda a derecha y en el eje X de arriba a abajo. La imagen siguiente muestra como se desplazan las coordenadas.

![Alineamiento de texto](/assets/docs/coordinates.png)

los casos especiales son las alineaciones de texto, en estos casos las coordenadas se marcaran como punto de inicio y la alineaciones dictaran para adonde debe moverse el texto.

## Bugs Conocidos
